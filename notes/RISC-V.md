# RISC-V

### 特权架构

|  lv  | encode |       name        | 模式 |
| :--: | :----: | :---------------: | :--: |
|  0   |   00   |     用户 User     |  U   |
|  1   |   01   | 管理员 Supervisor |  S   |
|  2   |   10   |   保留 Reserved   |  H   |
|  3   |   11   |   机器 Machine    |  M   |

### 通用寄存器

| 寄存器  | ABI 别名 | 说明                    |
| ------- | -------- | ----------------------- |
| x0      | zero     | 零, 忽略写入            |
| x1      | ra       | 返回地址                |
| x2      | sp       | 栈指针                  |
| x3      | gp       | 通用指针                |
| x4      | tp       | 线程指针                |
| x5      | t0       | 临时数据/备用链接寄存器 |
| x6~x7   | t1~t2    | 临时数据                |
| x8      | s0/fp    | 保存寄存器/帧指针       |
| x9      | s1       | 保存寄存器              |
| x10~x11 | a0~a1    | 函数参数/返回值         |
| x12~x17 | a2~a7    | 函数参数                |
| x18~x27 | s2~s11   | 保存寄存器              |
| x28~x31 | t3~t6    | 临时数据                |

### CSR 寄存器

**M 模式 CSR 寄存器**

| M 模式 CSR寄存器 |                    | 说明                           |
| ---------------- | ------------------ | ------------------------------ |
| mstatus          | 状态寄存器         | 保存全局中断使能状态和其他状态 |
| mtvec            | 异常入口基址寄存器 | 异常发生时需要跳转的地址       |
| medeleg          | 异常委托           |                                |
| mideleg          | 中断委托           |                                |
| mip              | 中断使能寄存器     |                                |
| mie              | 中断等待寄存器     |                                |
| hpm              | 硬件性能单元       | 性能计数                       |
| mhpmcounter      | 性能计数器         |                                |
| mhpmevent        | 配置性能事件       |                                |
| mcounteren       | 计数器使能         |                                |
| mcountinhibit    | 禁止计数           |                                |
| mscratch         |                    | 保存M模式指向hart上下文的指针  |
| mepc             |                    | 中断时, 当前PC值               |
| mcause           |                    | 保存发生中断和异常的情况       |
| mvtal            | 异常值寄存器       | 发生异常时, 保存出错的地址     |

**S 模式 CSR 寄存器**

S 模式 CSR 与 M 模式基本一样, 将第一个字母 m 改为 s

增加了 stap 寄存器, 保存页表基地址, 给 MMU 用
